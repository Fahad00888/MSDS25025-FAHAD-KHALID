## Project Name
**Sentiment Analysis of Amazon Product Reviews: Predicting Customer Emotions with NLP and ML**

## Author
Fahad Khalid  
GitHub: [Fahad00888](https://github.com/Fahad00888)

## Description
This project analyzes Amazon product reviews to predict customer sentiment using **Natural Language Processing (NLP)** and **Machine Learning (ML)** techniques.  
It includes:
- Dataset loading and exploration
- Data cleaning and wrangling
- Visualizations (histograms, bar charts, box plots, heatmaps)
- Machine Learning models for sentiment prediction
- Detailed report with screenshots

## Repository Contents
- Jupyter Notebook (`.ipynb`)
- Dataset file
- Word/PDF report
- Screenshots of all steps

## Purpose
To gain insights into customer emotions from product reviews and demonstrate the application of NLP and ML in real-world datasets.
